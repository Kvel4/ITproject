Действие: Заполнение анкеты
Пользователь вводит команду '/start' если заполняет анкету в первый раз и '/change' если хочет изменить ее
Программа спрашивает имя пользователя. Пользователь дает ответ'(принимается любой введенный текст)
Программа спрашивает пользователя о предпочитаемом типе ивентов. Пользователь дает ответ(принимается только ответ с предоставленной пользователю клавиатуры)
Программа спрашивает пользователя о городе его проживания. Пользователь дает ответ(принимается только ответ с предоставленной пользователю клавиатуры)
Программа спрашивает пользователя о наличии возрастного ограничения на ивенты. Пользователь дает ответ(принимается только ответ с предоставленной пользователю клавиатуры)
Программа спрашивает пользователя о его хобби. Пользователь дает ответ(принимается только ответ с предоставленной пользователю клавиатуры)
Анкета составлена


Действие: Желание узнать о функциях предоставляемых ботом
Пользователь вводит команду '/help'. Бот присылает сообщение содержащие краткое описание бота и доступные команды(так же с описанием их функционала)

Действие: Желание узнать об ивентах рядом с определенной станцией метро
Пользователь вводит команду'/subway'. Бот спрашивает станцию метро рядом с который вы хотите посмореть ивенты. Пользователь отвечает. Программ делает API запрос на сайт КудаГо и присылает сообщение состоящие из 5-20 ивентов с кратким описанием (в соответствии с анкетой и локацией).

Действие: Желание узнать об ивентах рядом с текущим местоположением
Пользователь вводит команду '/geo'. Бот запрашивает местоположение пользователя. Пользователь отвечает. Программ делает API запрос на сайт КудаГо и присылает сообщение состоящие из 5-20 ивентов с кратким описанием (в соответствии с анкетой и локацией).
